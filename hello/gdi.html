girl develop it

interactive experience design curriculum

As an instructor with the New York City girl develop it chapter, I've taught or assisted with many different front-end focused workshops. These classes included Intro to HTML, CSS and Javascript, Intermediate HTML, SS and Javascript, Advanced HTML, CSS and Javascript, Intro to Javascript and jQuery, Intermediate Javascript and jQuery, Intro to Git and Github, Web Concepts and Intro to Prototyping.

The first original curriculum I developed and taught was for a new Adobe tool freshly out of beta called Adobe XD (Experience Design). I designed it to showcase best practice design and prototyping features of a tool I believe to be exteremely relevant in the field. All of the curriculums are available online nationally.
